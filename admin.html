Role Definitions
Super Admin

Full access to everything — manage portal users, all data, system settings. Cannot be deleted by others.

Admin

Can manage users, assets and software inventory. Cannot manage other portal user accounts.

IT Manager

Full access to device/asset inventory. Read-only access to users and software.

Viewer

Read-only access to all pages. Cannot create, edit or delete any records.

The full token is shown once after generation. Store it securely — it cannot be retrieved again. A hint of the last 6 characters is shown above.
How to connect Google Workspace
1. Google Admin Console → Apps → Web and mobile apps → Add custom SAML app
2. In the Auto-provisioning section paste the SCIM Base URL and Bearer Token above
3. Map attributes: userName → Primary Email, name.givenName → First name, active → Suspended
4. Enable provisioning — TerzoCloud will automatically create / deactivate users on joiner / leaver events
