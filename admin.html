Role Definitions
Super Admin

Full access to everything — manage portal users, all data, system settings. Cannot be deleted by others.

Admin

Can manage users, assets and software inventory. Cannot manage other portal user accounts.

IT Manager

Full access to device/asset inventory. Read-only access to users and software.

Viewer

Read-only access to all pages. Cannot create, edit or delete any records.
